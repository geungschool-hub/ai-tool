# PRD: 현장체험학습 당일 공지 HTML 페이지

> 버전: v1.0 | 작성일: 2026-05-15

---

## 배경 및 목적

2학년 학생 179명이 **2026-05-28(목)** 현장체험학습 당일, 12개 계열별로 나뉘어 서로 다른 대학 및 체험 장소로 이동한다. 학생들이 당일 스마트폰으로 확인해야 할 정보가 분산되어 혼란이 예상되므로, **단일 HTML 파일**로 필요한 모든 공지를 집약한 모바일 친화적 안내 페이지를 제작한다.

기존 교직원 힐링연수 안내자료(`힐링연수_안내자료.html`)의 디자인 언어와 컴포넌트 구조를 재사용하되, 배색과 내용을 학생 대상 현장체험학습에 맞게 교체한다.

## 페이지 구성 (3 Page 구조)

### Page 1 — 메인 헤더 + 전체 일정

**헤더**
- 배지: `2026 · 혜화여고 2학년`
- 대제목: `현장체험학습`
- 부제: `2026년 5월 28일 (목)`
- 메타 정보 칩: 날짜 / 총 인원 / 계열 수

**섹션 A: 한눈에 보는 오늘 일정 (타임라인)**

```
08:30        등교 및 계열별 집합 (운동장)
09:00~09:15  학교 출발 (계열별 버스 탑승)
09:30~12:00  오전 — 대학 탐방 (계열별 대학 방문)
12:00~13:00  점심 식사 (각 장소 자체 해결)
13:00~16:30  오후 — 계열별 체험 활동
17:00~18:00  학교 귀교 (예정)
```

**섹션 B: 내 계열 찾기 카드 (인터랙티브)**
- 12개 계열을 카드 그리드로 표시 (계열명 + 인원 + 오전 대학 + 오후 장소)
- 카드 클릭 → 팝업 or 확장: 해당 계열 상세 정보 (체험 내용, 인솔 교사, 집합 위치 등)
- 힐링연수의 `dept-count-card` 패턴 재사용, 색상만 교체

---

### Page 2 — 집합 그룹 × 버스 배정

**섹션 C: 집합 장소별 그룹 안내**
- 5개 대학 집합 그룹을 카드로 구분
- 각 카드: 집합 장소(대학명) + 소속 계열 태그 + 인솔 교사명
- 버스 번호 배정 (미정 시 "확인 중" 표기)

**섹션 D: 오후 체험 장소 안내**
- 같은 장소를 이용하는 계열(예: 이과·바이오약학 → 서울시립과학관) 묶음 표시
- 장소명 + 주소 + 지도 링크 (클릭 시 네이버/카카오 지도 열기)
- 체험 시간 및 유의 사항 간략 표시

---

### Page 3 — 공지 사항 + 설문 안내

**섹션 E: 당일 주의 사항 (Notice Box)**
- 복장: 교복 착용 (체육·재활 계열은 체육복 허용 여부 별도 표기)
- 지참물: 교통카드 / 점심값 / 스마트폰
- 금지 사항: 무단 이탈 금지 / 음주·흡연 금지
- 귀가 전 인원 확인 절차

**섹션 F: 비상 연락처**
- 인솔 교사 계열별 연락처 표 (tel: 링크로 전화 연결)
- 학교 대표 번호

## 디자인 요구사항

### 색상 테마 (힐링연수 그린 → 학교 블루 계열로 변경)

```css
--primary-dark:  #0d47a1;   /* 짙은 파랑 (헤더, 강조) */
--primary:       #1565c0;   /* 기본 파랑 */
--primary-light: #bbdefb;   /* 연한 파랑 (배경 틴트) */
--accent:        #f57c00;   /* 오렌지 (CTA, 경고) */
--green:         #2e7d32;   /* 보조 색 (설문 버튼) */
--bg-page:       #d8e4f0;   /* 전체 배경 */
--bg-card:       #f4f8fd;   /* 카드 배경 */
```

### 레이아웃

- 힐링연수와 동일한 `.page` A4 카드 구조 (max-width: 780px)
- 모바일 우선: 카드 그리드 `repeat(2, 1fr)` → 모바일 `repeat(1, 1fr)` 반응형
- 버튼 탭 영역 최소 44×44px (터치 타겟)
- 폰트: Noto Sans KR (Google Fonts)

### 인터랙티브 요소

| 요소 | 동작 |
|------|------|
| 계열 카드 클릭 | 팝업 표시 (계열 상세 정보) |
| 지도 버튼 클릭 | 네이버 지도 앱/웹 링크 오픈 (새 탭) |
| 비상 연락처 전화번호 | `tel:` 링크로 바로 전화 연결 |
