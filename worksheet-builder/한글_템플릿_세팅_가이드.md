# 양식 B — 한글(HWP) 템플릿 세팅 가이드

> 한 번만 세팅하면 이후엔 템플릿 열고 내용만 채우면 됩니다.
> 한글 2018 이상 기준.

---

## 1단계 — 편집 용지 설정

`쪽 > 편집 용지` (또는 **F7**)

| 항목 | 값 |
|------|----|
| 용지 크기 | A4 |
| 위 여백 | 15mm |
| 아래 여백 | 12mm |
| 왼쪽 여백 | 14mm |
| 오른쪽 여백 | 14mm |
| 머리말 | 10mm |
| 꼬리말 | 8mm |

---

## 2단계 — 기본 글꼴 설정

`서식 > 스타일 > 바탕글` 편집

- 글꼴: **나눔고딕** (없으면 맑은 고딕)
- 크기: 10pt
- 줄간격: 180%

> 나눔고딕 미설치 시: https://hangeul.naver.com/font 에서 무료 설치

---

## 3단계 — 단락 스타일 등록

`서식 > 스타일` (또는 **F6**) → `새 스타일` 버튼

각 스타일을 아래 설정대로 만드세요.

## 4단계 — 빈칸 처리 방법

한글에서는 두 가지 방법 중 편한 걸 쓰세요.

### 방법 A — 공백 + 밑줄 (추천)
1. 빈칸 위치에 스페이스바 여러 번 입력 (xs: 4칸 / sm: 6칸 / md: 9칸 / lg: 13칸)
2. 그 공백들 드래그 선택
3. `U` (밑줄) 적용

**단축키 팁**: 밑줄 공백을 자동 교정에 등록해두면 편함
- `도구 > 자동 교정 > 자동 교정 내용` 에서 예: `[빈1]` → 밑줄 공백 6칸

### 방법 B — 선 그리기
- 빈칸 위치에 커서 두고
- `삽입 > 선` 또는 그리기 도구에서 가로선 삽입

---

## 5단계 — 표 스타일

표 삽입 후, 헤더 행 선택 → `표 > 셀 배경`

| 용도 | 배경색 |
|------|--------|
| 파랑 계열 헤더 | `#C5E3F7` |
| 초록 계열 헤더 | `#C3E9CC` |
| 일반 셀 | 없음(흰색) |

**셀 테두리**: 모두 `#BBBBBB`, 0.5pt

**글자**: 헤더 굵게, 가운데 정렬 / 일반 셀 가운데 정렬

---

## 6단계 — 머리말 설정

`삽입 > 머리말/꼬리말 > 머리말` → 양쪽

머리말 영역에서:

```
[왼쪽]                              [오른쪽]
생명과학  (13pt, 굵게, #1255A0)      학번 ______  이름 ______
II. 항상성 > 2-3  (7.5pt, #555555)
```

**머리말 아래 구분선**:
- 머리말 단락 선택
- `문단 모양 > 테두리` → 아래쪽만, 2.5pt, `#1255A0`

---

## 7단계 — 꼬리말 설정

`삽입 > 머리말/꼬리말 > 꼬리말`

- 가운데 정렬
- `삽입 > 쪽 번호` → `— {쪽번호} —` 형식으로 입력
- 글자: 9pt, `#555555`

---

## 8단계 — 템플릿으로 저장

`파일 > 다른 이름으로 저장`
- 파일 형식: **한글 서식 파일 (*.hwt)**
- 파일명: `양식B_학습지_템플릿.hwt`

저장 위치는 한글이 자동으로 템플릿 폴더를 제안합니다.

이후 `파일 > 새 문서 > 내 서식 파일`에서 불러올 수 있습니다.

---

## 사용 흐름 (세팅 완료 후)

1. 한글 열기 → `파일 > 새 문서 > 내 서식 파일 > 양식B_학습지_템플릿`
2. 머리말의 과목명·단원 수정
3. 본문에서 스타일 팔레트로 섹션 제목 클릭 → 내용 입력
4. 표 삽입, 내용 작성
5. 저장 후 인쇄 또는 PDF 내보내기
