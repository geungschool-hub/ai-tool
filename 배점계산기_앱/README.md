# 문항 정답 분포 · 배점 계산기

시험 출제 시 **정답 분포 균형, 배점 합계, 난이도-배점 정합성**을 한눈에 점검하고,
**상·중·하 문항 수와 총점만 입력하면 배점을 자동으로 계산**해 주는 웹 도구입니다.
설치·서버·인터넷 없이 **HTML 파일 하나로 동작**합니다.

---

## 1. 이 앱은 무엇인가요?

객관식·서답형 문항의 정답 번호, 난이도, 배점을 입력하면 다음을 실시간으로 검사·계산합니다.

- **정답 분포 검사** — ①~⑤ 정답 개수를 막대로 보여주고, 편차(최대−최소)가 허용폭을 넘으면 경고
  (예: 서울시교육청 지침 "20문항 기준 각 번호 ±2 → 편차 허용폭 4")
- **배점 합계 검사** — 객관식/서답형 배점 합을 목표 점수와 비교(부동소수점 오차 보정)
- **난이도-배점 침범 검사** — 낮은 난이도(하<중<상) 문항의 배점이 높은 난이도 배점을 넘으면 빨갛게 표시
- **자동 배점 계산** — 상·중·하 문항 수와 총점을 넣으면 하 ≤ 중 ≤ 상 순서를 지키며 합계를 정확히 맞춰 배점을 채움
- **여러 시험(버전) 관리** — 1학기 중간/기말, 2학기… 시험을 여러 개 저장하고 드롭다운으로 전환(수정 날짜 표시)

---

## 2. 주요 기능

### 입력
- **객관식**: 행마다 난이도(상·중·하) → 배점 → 정답(①~⑤) 입력
- **서답형**: 난이도 → 배점 입력 (정답 분포 계산에서는 제외)
- **키패드 빠른 입력**: 행을 클릭한 뒤 숫자 **1~5** 를 누르면 정답이 입력되고 **자동으로 다음 행**으로 이동
  (↑/↓·Enter 행 이동, 0·Backspace·Delete 지우기)

### 자동 배점
- **새로 구성**: 상·중·하 문항 수 + 총점 + 난이도 간격 → 표를 만들고 배점을 0.1점 단위로 배분, 합계를 총점에 정확히 일치
- **입력값 유지(체크 시)**: 이미 적은 난이도·배점·정답은 그대로 두고, **빈 난이도·빈 배점만 자동 완성** — 일부만 채워둬도 나머지를 채움

### 시험(버전) 관리 · 저장
- 상단 드롭다운으로 시험 전환, `＋새 시험 / ⧉복제 / 🗑삭제`
- **자동 저장**(브라우저 localStorage) — 별도 저장 버튼 불필요, "수정 6/16 14:30" 식으로 마지막 수정 시각 표시
- 제목을 직접 편집하면 시험 이름이 됩니다

### 내보내기 · 인쇄
- **⬇ 이 시험 JSON 내보내기** — 백업/공유용 `.json` 저장
- **⬆ JSON 불러오기** — 불러온 파일을 **새 시험(버전)으로 추가** (작년·지난 학기 파일을 새 버전으로 쌓기)
- **🖨 인쇄 / PDF** — 관리 바·버튼을 숨긴 깔끔한 출력

## 3. 사용 방법

1. `문항분포_배점_계산기.html` 파일을 **더블클릭**해 웹 브라우저(크롬/엣지 등)로 엽니다.
2. 상단 **설정**에서 객관식/서답형 목표 배점 합, 정답 편차 허용폭을 시험에 맞게 조정합니다.
3. **자동 배점** 패널에 상·중·하 문항 수와 총점을 넣고 `자동 배점 채우기` → 배점이 채워집니다.
   - 난이도·정답을 일부 먼저 입력했다면 **입력값 유지 ☑** 상태로 누르면 나머지만 채워집니다.
4. **객관식** 표에서 행을 클릭하고 키패드 1~5로 정답을 빠르게 입력합니다.
5. 하단 패널에서 분포·배점 합·난이도 정합성을 확인하고, 빨간 경고가 없도록 조정합니다.
6. 필요하면 **⬇ JSON 내보내기**로 백업하거나 **🖨 인쇄/PDF**로 출력합니다.

---

## 4. 데이터 저장 방식 (중요)

- 작업 내용은 **그 브라우저의 localStorage**에 자동 저장됩니다.
- 따라서 **다른 PC·다른 브라우저에서는 보이지 않습니다.** 옮기거나 공유하려면 **JSON 내보내기**로 파일을 저장한 뒤, 옮긴 곳에서 **JSON 불러오기** 하세요.
- 브라우저 캐시/사이트 데이터를 지우면 저장본이 사라질 수 있으니, 중요한 시험은 JSON으로 백업하세요.

## 5. 배포 / 등록 안내

이 앱은 **외부 라이브러리·서버가 전혀 없는 단일 HTML 파일**이라 옮기기 쉽습니다.

- **파일 업로드형 플랫폼**(예: 학교 자료실, 개인 허브): `문항분포_배점_계산기.html` 파일만 올리면 그대로 작동합니다.
- **링크(URL) 등록형**: GitHub Pages, Netlify Drop 등 무료 정적 호스팅에 파일을 올려 만든 주소를 등록하면 됩니다.
- **소스 붙여넣기형**: HTML 파일 내용을 그대로 복사해 붙여넣으면 됩니다.

> ※ 긍스쿨 허브 등록은 사용자가 직접 진행 예정. 위 형태 중 무엇이든 이 파일 하나만 있으면 됩니다.
