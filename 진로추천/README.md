# 진로·학과 매칭 상담 챗봇

진로가 불확실한 고등학생이 8개 질문(5~10분)에 답하면,
**계열 트랙 → 대표 학과 → 성적대별 추천 대학**까지 안내하는 CLI 챗봇이에요.

> **23개 트랙** 수록 — 환경(5) · 공학(4) · 의생명·보건(4) · 인문사회(5) · 법학(1) · 교육(2) · 예체능(2).
> 모든 대학·학과는 2026-06 기준 실제 대학 페이지에서 검증한 데이터입니다.

| 계열 | 트랙 |
|------|------|
| 환경 | 환경생태·복원 / 환경공학 / 대기·기후과학 / 환경·도시정책 / 환경보건 |
| 공학 | 기계·로봇 / 전기·전자 / 컴퓨터·소프트웨어 / 화학·신소재 |
| 의생명·보건 | 의예·의학 / 약학 / 간호 / 생명과학·바이오 |
| 인문사회 | 경영·경제 / 심리학 / 정치외교·국제 / 언론·미디어 / 어문·문학 |
| 법학 | 법학(학부) — ⚠ 로스쿨 미설치 대학만 학부 모집 |
| 교육 | 초등교육(교대) / 중등교육·사범 |
| 예체능 | 디자인·미술 / 체육·스포츠 — ⚠ 실기 전형 비중 큼 |

## 폴더 구조

```
진로추천/
├─ main.py                대화·추천 로직 (실행 파일)
├─ data/
│  ├─ majors.yaml         계열·학과·대학 매핑 데이터 (검증된 실데이터 + 출처)
│  └─ questions.yaml      8문항 질문 스크립트
└─ README.md
```

## 실행 방법

```bash
pip install pyyaml          # 최초 1회
python main.py
```

질문에 번호로 답하면 끝이에요. **외부 키·인터넷 없이 완전 무료로 동작**합니다.

### (선택) LLM으로 설명 다듬기

답변 설명을 더 자연스럽게 만들고 싶으면 Anthropic API 키를 설정하세요.
**키가 없으면 자동으로 무료 템플릿 출력**으로 동작하니, 설정은 선택입니다.

```bash
pip install anthropic
set ANTHROPIC_API_KEY=sk-...        # Windows (PowerShell: $env:ANTHROPIC_API_KEY="sk-...")
# 모델 바꾸려면(선택): set MAJORS_MODEL=claude-haiku-4-5   # 더 저렴
python main.py
```

LLM은 **뽑힌 2개 트랙의 데이터만** 받아 설명만 다듬어요(없는 학과·대학 생성 불가, RAG 구조).
모델 기본값은 `claude-opus-4-8`이고, 비용을 줄이려면 `claude-haiku-4-5`(가장 저렴)나
`claude-sonnet-4-6`으로 바꿀 수 있어요.

## 추천이 작동하는 원리

```
8문항 응답 → 키워드 추출 → 트랙 점수화(키워드 교집합 − 약한과목 감점)
          → 상위 2개 트랙 → 내신 등급대로 추천 대학 티어 선택 → 출력
```

- **두 파일을 잇는 계약**: `questions.yaml` 옵션의 `tags`(예: `자연관찰`, `수학`,
  `연구형`)와 `majors.yaml` 트랙의 `keywords`는 **같은 단어집**을 써야 매칭돼요.
- **내신(q8)**: `1~2등급→상위권`, `3등급→중상위권`, `4~5등급→중위권`,
  `모름→세 칸 모두` 보여줌.
- **약한 과목(q3)**: 그 과목이 핵심인 트랙에 감점.
- **지역(q7)**: 강한 필터가 아니라 안내용 가점(지방거점 선호 시 국립대 우선 안내).

## 데이터 확장 방법

### 새 학과·대학 추가 → `majors.yaml`만 수정

각 트랙은 아래 형태예요. **실제 대학 페이지에서 확인한 것만** 넣고,
못 찾은 자리는 `name: 확인 필요`로 두세요(임의 생성 금지).

```yaml
tracks:
  - id: 영문_식별자
    name: 트랙 이름
    college: 농생대 / 자연대        # 소속 단과대(같은 분야라도 단과대별로 다름)
    one_liner: 한 줄 설명(3줄 이내)
    keywords: [자연관찰, 과학, 연구형, ...]   # ← 질문 tags 와 같은 단어집
    majors: [대표학과1, 대표학과2, ...]        # 3~5개
    universities:
      top:      [{ name: 대학, college: 단과대, major: 학과 }]   # 상위권 1~2곳
      mid_high: [ ... ]                                          # 중상위권
      mid:      [ ... ]                                          # 중위권
    careers: [진로1, 진로2, 진로3]
    source: ["검증한 출처 URL", ...]           # 신뢰도 기록
```

### 새 질문·선택지 추가 → `questions.yaml`만 수정

옵션의 `tags`에 넣는 키워드는 위 `keywords`와 **철자까지 똑같이** 맞춰야 매칭돼요.
새 키워드를 만들면, 그 키워드를 반응시킬 트랙의 `keywords`에도 같이 추가하세요.

## 데이터 신뢰성 원칙

- 학과·대학명은 임의로 만들지 않음(hallucination 금지). `majors.yaml`이 유일한 출처.
- 검증 못 한 정보는 `확인 필요`로 표기.
- 모든 대학 데이터는 2026-06 기준 실제 대학 페이지에서 확인했고 `source`에 URL을 남김.
- 입결(상위/중상위/중위) 라벨은 대략적 분류 — 정확한 입결 백분위는 학교알리미
  데이터로 보강 예정(확장 과제).
