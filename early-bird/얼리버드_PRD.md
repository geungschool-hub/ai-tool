# 얼리버드 프로젝트 PRD

## 1. 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 얼리버드 프로젝트 (Early Bird Project) |
| 목적 | 아침 일찍 등교하는 학생들이 스스로 출석을 기록하고, 꾸준한 조기 등교를 장려하는 학급 특색 활동 |
| 사용 환경 | 교실 전자칠판 (터치 가능, 상시 표시) |
| 대상 | 2학년 7반 학생 26명 (20701~20726) |

---

## 2. 핵심 규칙

- **출석 인정 기준**: 매일 **07:40 이전** 체크 완료 시에만 유효
- 07:40 이후에는 버튼 잠금 처리 (체크 불가, 화면은 유지)
- 하루 중복 체크 불가 (같은 날 1회만 인정)
- 새 날이 되면 당일 체크 상태 초기화 (주간 누적은 유지)
- **주간 집계 기준**: 월요일~금요일 (주말 제외)

---

## 3. 화면 구성

### 메인 화면 (전자칠판 상시 표시)

```
┌─────────────────────────────────────────────────────┐
│  🐦 얼리버드 프로젝트         오늘: 2025-04-06 (월)  │
│                               ⏰ 현재 시각: 07:12    │
├───────────────────────────────┬─────────────────────┤
│                               │   오늘의 얼리버드    │
│  [ 01 권윤서 ]  [ 02 김민경 ] │                     │
│   이번 주 3회   이번 주 1회   │    ✅ 오늘 7명       │
│                               │    출석했어요!       │
│  [ 03 김민정 ]  [ 04 김시은 ] │                     │
│   이번 주 0회   이번 주 2회   │  ─────────────────  │
│                               │  ✔ 권윤서           │
│  ...                          │  ✔ 김채원           │
│                               │  ✔ ...              │
│                               │  ─────────────────  │
│                               │  [📋 전체 이력 보기] │
│                               │  [🔒 관리자]        │
└───────────────────────────────┴─────────────────────┘
```

### 구성 요소

#### 학생 카드 (좌측 메인 영역)
- 번호 + 이름 표시 (예: `01 권윤서`)
- 카드 하단: **이번 주 N회** 누적 횟수 (월~금 기준)
- 상태별 스타일:
  - 기본: 연한 배경 (터치 대기)
  - 오늘 체크 완료: 강조 색상(얼리버드 색상) + 체크 아이콘 ✔
  - 07:40 이후: 버튼 잠금 (비활성화 스타일, 체크 기록은 유지)

#### 우측 사이드패널
- 오늘 체크한 인원 수 (숫자 강조)
- 오늘 체크한 학생 이름 목록 (체크 순서대로)
- **[전체 이력 보기]** 버튼
- **[관리자]** 버튼

#### 상단 바
- 오늘 날짜, 요일
- 실시간 시계 (1초 단위 갱신)
- 07:40 이후: "오늘 얼리버드 마감! 내일 또 도전하세요 🐦" 메시지 표시

### 체크 시 응원 팝업
- 화면 중앙에 0.5초 등장 → 2초 유지 → 0.5초 사라짐
- 응원 문구: **"오늘도 화이팅! 🌟"** (추후 추가 예정)

---

## 4. 전체 이력 화면

- 메인 화면 위에 모달 또는 오버레이로 표시
- **구성**:
  - 날짜별 탭 또는 세로 목록 (최근 날짜 순)
  - 각 날짜별: 체크한 학생 수 + 이름 목록
  - 학생별 보기 전환: 특정 학생의 전체 출석 이력 조회 가능
- **닫기** 버튼으로 메인 화면 복귀

---

## 5. 관리자 기능

### 진입
- 우측 패널 **[관리자]** 버튼 터치
- 비밀번호 입력 화면 표시 → **0000** 입력 시 진입
- 잘못된 비밀번호 입력 시 "비밀번호가 틀렸습니다" 안내 후 재시도

### 관리자 화면 기능

| 기능 | 설명 |
|------|------|
| 수동 체크 추가 | 특정 날짜에 특정 학생 출석 수동 추가 |
| 체크 취소 | 잘못 기록된 출석 삭제 |
| 날짜 선택 | 오늘 이외의 날짜도 수정 가능 |
| 데이터 초기화 | 전체 또는 특정 날짜 데이터 초기화 (확인 다이얼로그 필수) |

- 관리자 화면에서도 전체 이력 조회 가능
- **[관리자 나가기]** 버튼으로 메인 화면 복귀

## 6. 데이터 구조

| 항목 | 저장 방식 | 설명 |
|------|----------|------|
| 일별 출석 | `localStorage` | 키: `earlybird_YYYY-MM-DD` → 값: `["20701","20703",...]` |
| 주간 누적 | 실시간 집계 | 해당 주 월~금 날짜 키를 조회하여 합산 |
| 주 초기화 | 자동 | 새 주(월요일) 감지 시 주간 카운트 자동 리셋 |

> **별도 서버/DB 없이 단일 HTML + localStorage로 동작**
> 브라우저 캐시 삭제 시 데이터 손실 주의 (전자칠판 전용 브라우저 권장)

---

## 7. 기술 사양

| 항목 | 내용 |
|------|------|
| 구현 형태 | 단일 `.html` 파일 (외부 리소스 없음) |
| 데이터 저장 | `localStorage` (브라우저 로컬) |
| 터치 지원 | iOS Safari / Android Chrome / 전자칠판 내장 브라우저 |
| 버튼 크기 | 학생 카드 최소 80×80px (전자칠판 터치 고려) |
| 폰트 크기 | 이름 최소 18px, 부가정보 최소 14px |
| 화면 비율 | 16:9 기준 전체화면 최적화 |
