# 작업노트 (USB 동반 메모리)

이 폴더는 **USB(`AI tool` 폴더)와 함께 다니는 메모리**입니다. 어느 PC에서 작업하든 `CLAUDE.md`가 이 폴더를 가리키므로, Claude가 관련 작업 시 해당 파일을 읽고 맥락을 이어갑니다.

- 자동 로드되는 진입점은 `..\CLAUDE.md` 입니다. 새 메모를 추가하면 이 README와 CLAUDE.md의 "작업 노트" 안내에 한 줄씩 적어 둘 것.
- PC에 묶인 Claude 자동 메모리(`C:\Users\...\.claude\...`)와 달리, 여기 파일들은 USB로 옮겨도 그대로 읽힙니다.
